# Calorie Counter Telegram Bot

Telegram-бот для учета калорий и продуктов.

## Возможности
- Регистрация пользователей
- Хранение индивидуальных данных
- Добавление и сохранение продуктов
- Удобный интерфейс через кнопки и меню
- Данные сохраняются между перезапусками

## Технологии
- Python
- Telegram Bot API
- JSON для хранения данных

## Статус проекта
Проект завершён. Поддержка и развитие не планируются.
